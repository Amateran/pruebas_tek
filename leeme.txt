

Entrega Prueba Tecnica.

RETO #1

Para el reto 1 el servicio se levanta de la siguiente forma
npm run dev

El servicio que corre cada 10 min se ejecuta de la siguiente forma

npm run servicio

Para revisar la lista con los filtros deben subir el server web
npm run dev
URL: http://localhost:4001/lista
